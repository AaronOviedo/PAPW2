Para crear una nueva rama con Git, se posiciona en la rama (veasé Master, por defecto) a clonar/copiar, se utiliza el comando... 
... "git checkout -b 'n' " ...
... dónde 'n' sería el nombre que se le quiera dar a la nueva rama, y ya se estaría posicionado en la rama creada, para hacer los cambios necesarios.

Adicionalmente para confirmar la creación de la rama, se utilizaría el comando...
... "git branch" ...
... para cerciorarse de que la rama fue creada correctamente.
